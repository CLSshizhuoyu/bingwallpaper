# Bing壁纸下载器 🖼️

一款高颜值、功能实用的Bing壁纸批量下载工具，支持自定义日期范围下载，自动保留图片版权元数据，附带紫色主题+楷体字体的精美UI界面。

## 软件简介

- **核心功能**：下载近7天内的Bing高清壁纸，支持单张/批量下载，自动写入图片标题和版权信息。

- **界面设计**：随机加载自定义背景图，紫色主题配色+楷体字体，适配背景尺寸的自适应窗口。

- **便捷特性**：单EXE独立文件，无需安装依赖，下载图片自动保存到软件所在目录，支持文件名去重。

## 快速开始

### 1. 环境要求

- 操作系统：Windows 7/8/10/11（32/64位）

- 无需安装Python，直接运行EXE文件即可。

### 2. 下载与运行

1. 获取软件包（包含 `Bing壁纸下载器.exe`）；

2. 将软件包解压到任意目录（确保目录有写入权限，避免C盘系统目录）；

3. 双击 `Bing壁纸下载器.exe` 启动软件。

### 3. 使用方法

1. 启动后，在输入框中输入下载参数：

2. 单张下载：输入 `0~7` 之间的数字（0=今日，1=昨日，以此类推）；

3. 批量下载：输入起始日期和结束日期，用英文逗号分隔（如 `0,3` 表示下载今日到3天前的壁纸）；

4. 点击「开始下载」按钮，进度条显示下载进度；

5. 下载完成后，图片自动保存到软件所在目录，文件名以日期命名（如 `20240520.jpg`）。

## 常见问题

### Q1：下载的图片保存在哪里？

A：图片默认保存到 **软件所在目录**，文件名以壁纸对应的日期命名（如 `20240520.jpg`），重复文件会自动添加序号（如`20240520_1.jpg`）。

### Q2：输入参数有什么限制？

A：日期范围仅支持`0~7`（近7天），批量下载时起始日期需小于等于结束日期（如 `2,5` 有效，`5,2` 无效），软件会自动修正超出范围的数值。

### Q3：下载失败/图片为空怎么办？

1. 检查网络是否正常，能否访问 [Bing官网](https://cn.bing.com)；

2. 确保软件所在目录有写入权限（可尝试右键以管理员身份运行）；

3. 避免在系统盘（C盘）的Program Files等受保护目录运行。

### Q4：背景图/图标不显示怎么办？

A：确保 `bj1.jpg~bj3.jpg` 和 `icon.ico` 与EXE文件在同一目录，且文件格式正确（图片为JPG格式，图标为ICO格式）。

## 开发与打包（进阶）

### 依赖库

```Bash
pip install requests pillow pyqt5 pyinstaller
```

### 打包命令

将源码、背景图、图标放在同一目录，执行以下命令打包为单EXE：

```Bash
pyinstaller -F -w -i icon.ico --add-data "bj1.jpg;." --add-data "bj2.jpg;." --add-data "bj3.jpg;." --add-data "icon.ico;." --hidden-import PyQt5.sip --hidden-import PIL --encoding utf-8 main.py
```

### 打包参数说明

|参数|作用|
|---|---|
|`-F`|打包为单个EXE文件|
|`-w`|隐藏控制台窗口（GUI模式）|
|`-i icon.ico`|设置程序图标|
|`--add-data "文件;."`|打包资源文件到EXE|
|`--hidden-import`|补全隐式依赖|
|`--encoding utf-8`|解决中文乱码问题|
